Some Tags

Tags that go in body

<p> Paragraph: holds text like <p> Hi </p>

<a> Link/Anchor: link to another webpage like <a href="https://www.google.com">Link Text</a>

The part with href="URLGOESHERE" is called an attribute.

<img> Image: <img src="rumakerspacelogo.gif"> for local images and a url instead for other images

<div> Division: groups HTML like: <div> html goes here </div>

All tags can have an id="" and class="" attribute that we can use to identify them in other languages

<h1> Heading </h1> Depending on the number (from 1 to 5) the heading is smaller or larger (1 is largest)

<b> bold text</b>

<u>underlined text</u>

<s>striked through text</s>

<br> New Line

Tags that go in head

<script> Javascript Code goes here</script>

<style> CSS Code goes here</style>

To Nest Tags:

<h4><b>Bold Heading</b></h4>

Output:

Bold Heading

Styling Your HTML with CSS

Say you have a tag like:

<div id="div1"></div>

This is empty so it looks like:
However, in our head, we will add a style:

<style>

#div1{

width: 300px;

height: 300px;

background-color: rgb(255,0,0);

}

</style>

Now our div looks like:

The #div1 is a selector, it selects any tag that has id="div1"

Within the curly braces, we can put in what are called css properties that modify whatever tag is selected. They are written as follows:

property-name: value;

There are many properties including ones that space text, change fonts, and pretty much anything you can think of. A good refrence of all these properties and examples can be found here:

https://www.w3schools.com/cssref/default.asp

Animating your CSS

Css properties don't have to be static, they can change with time using CSS animations.

To create an animation we can edit our origrinal styling:

#div1{

width: 300px;

height: 300px;

background-color: rgb(255,0,0);

animation: anim 3s infinite alternate;

}


@keyframes anim{

0% {background-color: #384ab3;width:300px;}

25% {background-color: #c7bf3d;}

50% {background-color: #39fe94;}

75% {background-color: #e485cf;}

100% {background-color: #475b40;width:100px;}

}


And now our div looks like this:

The property animation sets out how the animation will play out, it is saying to use the keyframes called anim, have a duration of 3 seconds, run infinitly many times, and play it forwards and reverse each time (if you only want to only play forward and then jump back to 0% after it hits 100% in the keyframes, just get rid of alternate).

Using HTML and CSS you can do all sorts of crazy things!!!
